Requerimientos:
Asegúrese de tener instalado Python y las siguientes bibliotecas antes de ejecutar el código:

pandas
seaborn
matplotlib
numpy
scikit-learn

Instalación:
pip install pandas seaborn matplotlib numpy scikit-learn

Datasets utilizados:

1. MNIST (Modified National Institute of Standards and Technology database)
-Fuente: https://en.wikipedia.org/wiki/MNIST_database

2. KMNIST (Kuzushiji-MNIST)
-Fuente: https://github.com/rois-codh/kmnist

Instrucciones de ejecución

1. Instalación datasets
-Descargar los datasets MNIST y KMNIST desde los links dados
-Cree una carpeta llamada 'datos' en el mismo directorio donde se encuentra el archivo principal.
-Coloque dentro de esa carpeta los archivos correspondientes a los datasets MNIST y KMNIST 

2. Ejecutar el código
-Ejecute el script desde la terminal ubicado en el mismo directorio que este README.txt con:
python TP-02-burritodiabolico.py
